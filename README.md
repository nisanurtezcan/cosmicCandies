# Cosmic Candies: A Universe of Us
A simple universe created for a small game jam. Pixel art assets were made in Aseprite.

**LIVE LINK:**  
https://ninutt.itch.io/cosmic-candies-a-universe-of-us 

If you enjoy cozy games and pixel-art assets, feel free to follow my itch.io page 🌸  
https://ninutt.itch.io/

This project was created for the **Create A Universe** game jam.
I didn’t have much time to develop it, but it turned into a cute little experiment where I could learn a few new things and take a short creative break from university work.
In total it took around 5–6 hours, so it’s a very small project, but a fun one.

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Aseprite (for pixel art and small animations)

---

## 🎮 Concept

Welcome to the **Cosmic Candies Universe**!
Here you will meet three sweet cosmic friends:
* **Candy Cotton**
* **Caramel Pack**
* **Rainbow Lolly**

You can click on the planets to discover their personalities and little stories.
A small reminder from this tiny universe:
**everyone shines in their own way, our differences are what make us unique.**

---

## 📚 What I Practiced

* Creating simple web animations using **`setInterval()`**
* Using **forEach loops** for interactive elements (opening and closing planets)
* Practicing **DOM manipulation**, especially with `id` selections
* Using `classList` functions like **add / remove / hidden / visible**
* Adding small **CSS hover animations** and positioning elements
* Creating **short pixel animations in Aseprite** instead of only static images

---

## 🔧 What Can Be Improved

* There are still some **responsiveness issues** on different screen sizes
* Animations could be **smoother**, especially the background picture could be done better
* The experience is very short — more **interactive elements** could be added
